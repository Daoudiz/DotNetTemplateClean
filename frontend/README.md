## 🇬🇧 English

This project is the user interface (Frontend) of a management application, developed with **Angular 21** and the **CoreUI** template.

### 🛠 Technologies Used
* **Angular**: Core framework (v21).
* **CoreUI**: UI component template (Bootstrap 5).
* **RxJS**: Asynchronous data stream management.
* **jwt-decode**: JWT token decoding.
* **Signals**: Reactive state management.

### 📋 Prerequisites
* [Node.js](https://nodejs.org/) (v21+)
* [Angular CLI](https://angular.io/cli) (`npm install -g @angular/cli`)

### ⚙️ Installation and Setup
1. **Clone the repository:**
   ```bash
   git clone [https://github.com/Daoudiz/DotNETSampleArch.AngularFront](https://github.com/Daoudiz/DotNETSampleArch.AngularFront)
   cd DotNETSampleArch.AngularFront
   npm install
    ng serve

### 🔗 Backend Integration

    Backend Repository: https://github.com/Daoudiz/DotNETSampleArch

    Default API URL: https://localhost:7270/api

    Auth: JWT tokens stored in localStorage.

##  🇫🇷 Français

Ce projet est la partie interface utilisateur (Frontend) d'une application de gestion, développée avec Angular 21 et le template CoreUI.
🛠 Technologies utilisées

    Angular : Framework principal (v21).

    CoreUI : Template de composants UI (Bootstrap 5).

    RxJS : Gestion des flux de données asynchrones.

    jwt-decode : Décodage des tokens JWT.

    Signals : Gestion de l'état réactif.

### 📋 Pré-requis

    Node.js (version 21 ou supérieure)

    Angular CLI (npm install -g @angular/cli)

### ⚙️ Installation et Lancement

    Cloner le repository :
    Bash git clone [https://github.com/Daoudiz/DotNETSampleArch.AngularFront]
cd DotNETSampleArch.AngularFront

Installer et Lancer :
Bash

    npm install
    ng serve

### 🔗 Intégration Backend

    Dépôt Backend : https://github.com/Daoudiz/DotNETSampleArch
